# Eid Gift Card Creator

A modern interactive web app for designing personalized Eid gift cards with live preview, drag-to-position text, template selection, and quick export options.

## Overview

Eid Gift Card Creator lets users customize beautiful greeting card templates in real time. Users can choose a template, add Bangla or English text, change font styles, adjust size and color, move the text directly on the card, and download or share the final design.


## Features

- Live card preview
- Multiple gift card templates
- Bold and italic text styling
- Drag and place text anywhere on the card
- Flip-card preview interaction
- Download card as PNG
- Share card quickly

## Tech Stack

- HTML
- CSS
- JavaScript
  

## Why I Built This

The goal was to make card editing feel simple, elegant, and visually engaging while keeping the interface modern and user-friendly.



## Getting Started

1. Clone the repository
2. Open the project folder
3. Run the app in a local environment
4. Start customizing your Eid card
